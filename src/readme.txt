# S21_SmartCalc_1.0

Проект S21_SmartCalc_1.0 представляет собой калькулятор, реализованный на языке программирования C с использованием структурного подхода. Он имеет следующие основные функциональности:
1. Вычисление арифметических выражений с учетом приоритетов операций. Это означает, что калькулятор способен правильно обрабатывать выражения с умножением, делением, сложением и вычитанием, придавая приоритет более высоким операциям.
2. Поддержка математических функций, таких как синус, косинус, логарифм и другие. Это позволяет пользователю вычислять более сложные математические функции.
3. Использование переменной x. Калькулятор может работать с переменной x в выражениях, что позволяет пользователю вычислять выражения, содержащие переменные.
4. Построение графика функции. Калькулятор может строить график функции, заданной пользователем, что может быть полезно для визуализации математических функций.
5. Кредитный и депозитный калькулятор. Проект также включает в себя кредитный и депозитный калькулятор, который может помочь пользователям рассчитать ежемесячные платежи, общую сумму кредита, проценты и другие параметры для кредитов и депозитов.
